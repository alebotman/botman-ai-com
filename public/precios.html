Precios - Codacle Migration Service

Nuestra estructura de precios (*) es simple y directa. Sin costos ocultos. Ofrecemos una garantía de devolución de dinero si la aplicación migrada no coincide con la funcionalidad exacta de la aplicación Delphi original.

Estructura de Precios

  • Precio Fijo Único de Engagement y Configuración: USD 15,000
  • Precio por Formulario: USD 1,000 por formulario migrado de Delphi al framework destino (con un mínimo de 100 formularios)
  • Adaptación Opcional a Diferente Stack Tecnológico (ej: Node + Angular): Tarifa única adicional de USD 18,000 (**)
  • Consultoría Opcional / Tarifa de Desarrollo Personalizado por Hora: USD 90 por hora

Las personalizaciones adicionales como integraciones de API, herramientas de documentación especializadas u otros requisitos únicos se cotizarán por separado según el esfuerzo estimado.

Estos precios asumen que el modelo de base de datos permanece igual y cualquier objeto de base de datos (como procedimientos almacenados) también permanece funcionando como parte de la aplicación migrada. Cualquier cambio adicional a estos elementos se cotizará por separado.

(*) Los precios indicados no incluyen los impuestos estatales o locales aplicables. El impuesto sobre las ventas se agregará en las jurisdicciones donde lo exija la ley. Todos esos impuestos se agregarán donde corresponda.

(**) Cualquier personalización de nuestro producto principal permanece como propiedad de BotMan LLC.

Responsabilidades del Cliente

Para garantizar un proceso de migración fluido, requerimos que los clientes:

  • Describan claramente los estándares y preferencias tecnológicas, incluidas las convenciones de codificación, estructuras de carpetas, versiones de bibliotecas o cualquier otra guía relevante.
  • Proporcionen acceso seguro a sus repositorios de código y bases de datos de prueba.
  • Especifiquen las preferencias de entrega.
  • En caso de que se requieran pruebas funcionales por nuestra parte, proporcionen descripciones detalladas de las pruebas requeridas para cada formulario, incluidos los datos de prueba disponibles y el acceso a un entorno de prueba adecuado.

Garantía y Políticas

Prueba de Concepto (POC): No ofrecemos POCs de forma gratuita; sin embargo, proporcionamos una garantía de devolución de dinero de 60 días si la aplicación migrada no coincide con la funcionalidad exacta de la aplicación Delphi original.

Funcionalidades Específicas de Escritorio: Las pantallas migradas replicarán de cerca la apariencia y funcionalidad de las pantallas originales, excepto los comportamientos específicos de escritorio que pueden requerir adaptación para entornos web (ej. interacción con disco local).

Para discutir cómo Codacle puede optimizar la modernización de su aplicación, contáctenos en info@botman-ai.com.
