LangGraph + Gemini Pro + Custom Tool + Streamlit = Multi-Agent Application Development

En este artículo, se explica cómo crear un chatbot utilizando LangGraph, Gemini Pro, Custom Function y Streamlit para responder a solicitudes de soporte de los usuarios. LangGraph es una biblioteca diseñada para construir aplicaciones multiactor con estado utilizando modelos de lenguaje de gran tamaño (LLM, por sus siglas en inglés).

Mediante la extensión del Lenguaje de Expresión LangChain, se facilita la colaboración cíclica de múltiples cadenas (o actores) a lo largo de varios pasos, permitiendo describir e introducir ciclos necesarios para el agente, lo cual no era sencillo con el Lenguaje de Expresión LangChain debido a su inadecuación para describir ciclos.

Además, LangGraph simplifica la introducción de ciclos en aplicaciones LLM, aumentando el valor de "LangChain" al permitir la creación fácil de cadenas personalizadas.
